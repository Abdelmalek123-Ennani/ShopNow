### installation :
fetch the project 
```bash
  git clone https://github.com/Abdelmalek123-Ennani/ShopNow.git
```
change the directory to directory project : 
```bash
  cd ShopNow
```
install missing dependencies : 
```bash
  npm install
```
run react project : 
```bash
  npm start
```
